# QA Automation E-commerce Project

## 🚀 Technologies
- Python
- Selenium
- Pytest
- Page Object Model (POM)

## 🧪 Test Scenarios
- Successful login
- Add product to cart
- Cart validation

## 📌 Project Description
This project automates a real e-commerce user flow using Selenium and Python. It follows the Page Object Model (POM) design pattern to ensure scalability and maintainability.

## ▶️ How to run
1. Install dependencies:
pip install -r requirements.txt

2. Run tests:
pytest

## 💡 Purpose
This project was built as part of a QA Automation portfolio to demonstrate real-world testing skills.
